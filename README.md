# 💪 RKFit Nonstopes – Daily Workout Notifier

A modern web-based workout reminder application that helps users stay consistent with their fitness routine by scheduling smart browser notifications.

---

## 🚀 Features

- 🔔 Browser-based workout notifications  
- 📅 Schedule reminders for specific days and times  
- 🏋️ Select different workout types (Strength, Cardio, Yoga, etc.)  
- ⚡ Set urgency levels (Normal, High Energy, Chill)  
- 💾 Data stored locally using LocalStorage  
- 🧪 Test notification feature  
- 🎨 Clean and responsive UI with modern design  

---

## 🛠️ Technologies Used

- **HTML5** – Structure  
- **CSS3** – Styling & animations  
- **JavaScript (Vanilla JS)** – Functionality  
- **Notification API** – Browser notifications  
- **LocalStorage API** – Data persistence
